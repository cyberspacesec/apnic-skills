欢迎使用CNNIC IP地址注册信息查询系统
您可以查到CNNIC及中国大陆的IP地址分配信息

IPv4地址查询 请输入IPv4地址(如210.72.0.0 或 210.72.0.0/19 或 210.72.0.0 - 210.72.31.255)，查询IPv4地址的相关信息
联系人信息查询 请输入联系人账号(如IPAS1-CN)，查询联系人相关信息
IPv6地址查询 请输入IPv6地址(如2001:F88::或 2001:F88::/32)，查询IPv6地址相关信息
AS号码查询 请输入AS号码(如9811或AS9811)，查询AS号码相关信息
路由注册查询 请输入地址(如211.144.211.0/24或121.58.144.0/20)，查询路由注册相关信息
网络名称查询 请输入网络名称(如CSTNET)，查询网络名称相关信息
